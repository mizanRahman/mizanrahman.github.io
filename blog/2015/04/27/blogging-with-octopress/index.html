Blogging With Octopress

Installation instructions

git clone git@github.com:mizanRahman/mizanrahman.github.io.git
git checkout -b source origin/source
rake setup_github_pages

How to run

rake preview

create new post

rake new_post["title"]

deploy/publish your change

rake genrate
rake deploy

commit changes to source branch and push to the repository

git add .
git commit -m 'your message'
git push origin source

More help:

  • http://octopress.org/docs/blogging/
  • http://octopress.org/docs/deploying/github/
